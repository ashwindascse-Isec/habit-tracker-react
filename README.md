# Habit Tracker App

A React-based habit tracking application that helps users build consistency through daily tracking and streaks.

## Features
- Track daily habits
- Streak system for motivation
- Weekly heatmap visualization
- Add and delete habits
- Data persistence using localStorage

## Tech Stack
- React (Vite)
- JavaScript
- CSS

## Run Locally
```bash
npm install
npm run dev
```

## Future Improvements
Backend integration (Node.js + DB)
User authentication
Advanced analytics
